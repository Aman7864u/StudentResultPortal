Core Features

Everything You Need

Built with Spring Boot 3.2, Spring Data JPA, H2/MySQL, Lombok, and Springdoc OpenAPI for a complete academic management system.

🎓
Student Management

Complete CRUD for student profiles. Search by name, roll number, or department. Tracks CGPA and subject count automatically.

📚
Subject Catalogue

Manage subjects with unique codes, credit hours, department, and semester mapping. Prevents duplicate codes with validation.

📊
Auto Grade Calculation

GradeCalculator utility auto-computes letter grades (O to F), 10-point grade points, percentages, and PASS/FAIL status.

🔌
REST API + Swagger

Full RESTful endpoints with CORS enabled. Interactive Swagger UI at /swagger-ui.html for testing without Postman.

🛡️
Global Exception Handling

Consistent JSON error responses for 404 Not Found, 400 validation errors, and 500 server errors via @RestControllerAdvice.

🗄️
Dual Database Support

Ships with H2 in-memory DB for instant start. Switch to MySQL with a single config block — no code changes needed.

24 Endpoints Ready
